What is ANE-Training?

Your Mac has a Neural Engine (ANE) — an AI chip with 16 cores and up to 38 TOPS. Apple officially only allows inference (CoreML). This project breaks that restriction: training directly on the ANE.

Key Differentiators
◆

Dynamic Spatial Packing

Weights as IOSurface input instead of BLOBFILE. Compile once, change weights as often as you want. No recompile limit.

★

76+ Private API Classes

Discovered via runtime introspection. Other projects know only 4. Version detection survives Apple API changes automatically.

{}

Stable C API (libane)

73KB, usable from any language. Your code against ane.h never breaks, no matter what Apple changes.

●

100% Local, 100% Private

No cloud service, no account, no data ever leaves your device. Train models with complete privacy.
